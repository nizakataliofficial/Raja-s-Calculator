# Simple Web Calculator

A clean, modern, and fully responsive web calculator built from scratch using HTML5, CSS3, and JavaScript (ES6).

## Features
- **Core Arithmetic Operations:** Supports addition, subtraction, multiplication, division, and percentage calculations[cite: 1, 3].
- **Instant Functions:** Includes `AC` (All Clear) to reset the display and `DEL` to delete the last character[cite: 1, 3].
- **Responsive Layout:** Designed with a glassmorphism style and a dark semi-transparent UI that centers beautifully on all screen sizes[cite: 2, 3].
- **Smooth Interaction:** Features dynamic hover effects on buttons for an enhanced user experience[cite: 2].

## Tech Stack Used
- **HTML5:** For semantic structure[cite: 3].
- **CSS3:** For styling, centering layouts (Flexbox), custom font handling, and hover animations[cite: 2].
- **JavaScript (DOM Manipulation):** Using `querySelectorAll`, `Array.from()`, and event listeners to dynamically handle button clicks and calculate results smoothly[cite: 1].
